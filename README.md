# Ldplayer_multi_controler
LD플레이어 여러개 동시 제어

<img width="303" height="108" alt="image" src="https://github.com/user-attachments/assets/d714ba62-40fe-4b0b-9200-db308d1958f2" />
<p></p>
<p>본 프로그램은 오토핫키v1 으로 제작된 ld플레이어 멀티 컨트롤러 입니다.</p>
<p>여러개의 ld플레이어를 한번에 제어할 수 있습니다.</p>
<p>해상도 및 창 크기가 동일하다면 키보드, 마우스를 동일한 입력을 줄 수 있습니다.</p>
<p>특수키, 마우스 우클릭은 지원하지 않습니다.</p>
